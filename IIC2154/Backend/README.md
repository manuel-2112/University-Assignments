# Backend Grupo 5 Capstone

## Requisitos
* Pedir .env.local a integrante del backend vía whatsapp

## Cómo correrlo con Docker 
1. Ir a branch *develop*
2. docker compose up --build
3. Ingresar a http://localhost:8080/api/health para verificar que este corriendo

## Link postman documentación:
* https://app.getpostman.com/join-team?invite_code=192e0222b92b6647de130de6984736a2
* En este link se pueden ver ejemplos de los distintos requests
* Seleccionar el environment Development, para poder correr las distintas requests
* En el workspace *Grupo 3 Backend*, presionar los tres puntos, bajar hasta encontrar View Documentation
* Al hacer View Documentation pueden ingresar a la documentación de los distintos requests
  
## Extra
* Cualquier duda contactarse con alguien de backend, sobre documentación, problemas con correr API, entre otros.
